# Note Vault Debugging Challenge

## Setup Steps
1. Run `npm install`
2. Run `npm test` to see the failing tests.

## Hints
1. Double-check the order of operations and arguments when validating user credentials.
2. Are you sure your database interactions are completing before you move on? Also, verify how you refer to a document's unique identifier.
3. A secret key with no substance isn't much of a secret, is it?
